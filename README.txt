Readme

M_*.txt: Read counts on selected *mutant* alleles for cells post filtering. Mutations are in columns, cells in rows.
N_*.txt: Read counts on selected *reference* alleles for cells post filtering Mutations are in columns, cells in rows.

*: Patient ID. See Overview.ppt for a bit more info on the two patients

celltypes.rda: R Data object containing two names vectors that associate each cell (row from the count matrices) with a cell type
